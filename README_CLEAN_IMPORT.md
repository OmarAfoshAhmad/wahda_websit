# خطوات تنظيف وإعادة الاستيراد

1. شغّل السكريبت التالي لإلغاء جميع حركات الاستيراد:

   ```bash
   npx ts-node scripts/cleanup-import-transactions.ts
   ```

2. بعد التأكد من أن جميع الحركات أُلغيت (is_cancelled=true)، نفّذ عملية الاستيراد من الملف من جديد.

3. راقب النتائج وتأكد أنه لا يوجد أي تكرار في حركات الاستيراد.

> هذه الطريقة تضمن توزيع نظيف وصحيح 100%.
